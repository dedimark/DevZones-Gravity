# DevZones-Gravity
 
### EN
Retrieves the gravity value you specify in the zone name that falls into the "x.x_gravity" zone.

### TR
"x.x_gravity" bölgesine giren bölge adında belirttiğiniz yerçekimi değerini alır.

0.5_gravity
1.0_gravity
